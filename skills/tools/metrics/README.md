# Metrics Skill

Write and query numeric metrics. Latency, throughput, error rates, time-series.

## Capabilities

- **write**: Write metric points
- **query**: Query metrics (time range, aggregation)
- **list**: List metric names

## Pip Packages

None. Writes/reads Sophon SQLite metrics table.

## Role

- Record performance data
- Use with troubleshoot for latency/throughput analysis
