# Log-Analyze Skill

Query and analyze Sophon application logs. Errors, exceptions, system events.

## Capabilities

- **list**: List log sessions and dates
- **query**: Query raw log entries (level, keyword, regex)
- **analyze**: Aggregate by level, time for charts

## Pip Packages

None. Reads Sophon SQLite logs table.

## Role

- Troubleshoot errors and exceptions
- Use with troubleshoot for diagnostics
