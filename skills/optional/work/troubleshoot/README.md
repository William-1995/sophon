# Troubleshoot Skill

System anomaly and error diagnosis. Logs, traces, metrics, root-cause analysis.

## Capabilities

- **diagnose**: Full diagnostic, aggregates traces and logs, returns summary and chart data

## Dependencies

- `log-analyze`, `trace`, `metrics`, `memory`

## Pip Packages

None. Uses Sophon built-in data.

## Role

- Error localization
- Latency and performance analysis
- Frontend renders charts from returned data
